# AresMUSH Plugin Development

When developing this plugin or related plugins (Grimoire, Inklings), refer to the
AresMUSH Plugin Development Guide:

**https://github.com/MischiefMaker/ares-inklings-plugin/blob/main/ARES_PLUGIN_DEVELOPMENT_GUIDE.md**

This guide captures conventions, architecture, and lessons learned from the Inklings
project and applies to any AresMUSH plugin. It covers:

- Research process and verification against actual Ares source
- Web portal conventions (GameApi contracts, component patterns)
- Ruby plugin structure and conventions
- Installation best practices
- Common mistakes and how to avoid them
- Complete review checklist

Start here for any plugin task—new feature or bug fix.
