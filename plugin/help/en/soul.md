# SOUL

SOUL is a story-first character progression system: Aspects and Skills,
optional Resonance, XP-driven advancement, Boons & Banes, Culminations, and
configurable rolls with GM-assisted review.

SOUL is under active development. Character commands, rolls, and the Boon &
Bane catalogue will appear here as each subsystem is implemented - see
`docs/spec/ROADMAP.md` in the plugin repository for the current build phase.
